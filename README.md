# Myntra Home Page UI Design

## Overview

This project is a UI design recreation of the Myntra home page created using Figma. The objective was to practice modern e-commerce interface design, visual hierarchy, typography, spacing, and layout organization while studying UI/UX design principles.

The design focuses on creating a clean and user-friendly shopping experience inspired by Myntra's interface.

## Project Goals

* Practice UI design fundamentals
* Improve visual hierarchy and spacing
* Understand e-commerce page layouts
* Create a clean and intuitive user interface
* Build a beginner UI/UX portfolio project

## Design Process

1. Researched Myntra's layout and structure
2. Planned the content hierarchy
3. Designed the interface in Figma
4. Refined spacing, typography, and visual consistency
5. Created an interactive prototype
